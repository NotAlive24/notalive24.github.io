23. Cloud Characteristics

Cloud-based assets
Cloud-based assets are resources an organization uses through cloud computing. Cloud computing provides on-demand access to computing resources from almost anywhere. These resources are usually highly available and scalable, and organizations typically lease them from a cloud provider.

Benefits of cloud computing
Cloud usage is metered, so cost is based on what is consumed, and charges can be tracked by department or function. It reduces cost of ownership because the organization does not need to buy and maintain as much hardware or support infrastructure. Cloud can also reduce energy and cooling needs through more efficient resource use. It allows fast scaling of software and data services without installing large amounts of local hardware.

Main Idea
Cloud assets are leased, scalable resources accessed on demand, and they save cost while improving flexibility and scalability.
