17. Security of the Network

Networks must be protected because protocols like TCP/IP have many vulnerabilities.

If TCP/IP is implemented poorly, networks can be exposed to attacks such as DoS/DDoS, fragment attacks, oversized packet attacks, spoofing, and man-in-the-middle attacks.

Networks are also vulnerable to passive attacks such as monitoring or sniffing, where attackers watch traffic to gather information.

Because of these risks, network security requires controls that protect against both active attacks and passive observation.

Main Idea
Network security is necessary because TCP/IP and network traffic can be attacked, exploited, or monitored.
