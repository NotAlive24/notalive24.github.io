1. Data Handling

Data Handling (Data life cycle) – the stages data goes through from creation to deletion.

Stages

  • Create (making new data or information)
    Data is first produced, such as writing a document or collecting data.
  • Store (saving data somewhere)
    Data is saved on devices like databases, servers, or storage systems.
  • Use (accessing and working with data) Data is viewed, edited, analyzed, or processed by users or applications.
  • Share (sending or giving data to others)
    Data is transferred between people, systems, or locations.
  • Archive (storing data long-term but not actively using it)
    Data is kept for records or compliance but rarely accessed.
  • Destroy (permanently deleting data)
    Data is removed when it is no longer needed.
