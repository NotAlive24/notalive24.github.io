2. Hack your first machine

In this task we will be using tool like Gobuster, to find hidden directories of a bank site and transfer money to another account.
First, we should start the VM and only it's on, open the terminal.
Then we run gobuster on the terminal like 

gobuster dir -u http://fakebank.thm -w wordlist.txt
Then we transfer $2000 from account 2276 to account 8881 That's it for the task given.
